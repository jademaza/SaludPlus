#  SaludPlus

Sistema web de gestión de pacientes desarrollado en Java utilizando JSP, Servlets y el patrón de arquitectura MVC.

---

##  Descripción

SaludPlus es una aplicación web que permite registrar pacientes y visualizar su información de manera organizada, proporcionando una interfaz sencilla y fácil de usar.

Este proyecto fue desarrollado con fines académicos para fortalecer conocimientos en desarrollo web con Java.

---

## Funcionalidades

- Registro de pacientes
- Visualización de datos registrados
- Navegación entre páginas JSP
- Separación de responsabilidades mediante MVC
- Interfaz web con HTML y CSS

---

## Tecnologías utilizadas

- Java
- JSP (JavaServer Pages)
- Servlets
- HTML5
- CSS3
- Eclipse IDE
- Git
- GitHub
